INCL - Include an external file (Macro-80C)

[label]	INCL	filename[.ext]

Uses source code from the source file given by filename into the current source file during assembly. Several directives have been included to provide legacy support for a wide range of assemblers. See INCLUDE for more information.


Example: 

	incl	test2.asm


See also:
INCLUDE, USE, LIB
