INCLUDEBIN - Include binary file

[label]	INCLUDEBIN	filename[.ext]

Includes in any file reguardless of type, starting at the current program counter location. The contents of the file will not be processed by the assembler but will be inserted into the assembled file without modification. These directives are handy for directly including raw binary data rather than first converting it to fcb statements and then assembling it.


Example: 
	includebin	test.bin
