ZMQ - Zero constant zero quad bytes

[label]	ZMQ	expression

Allocates a block of quad bytes (32 bits). Each quad byte is assigned the initial value of zero. The number of quad bytes allocated is given by the expression in the operand field. If the expression contains symbols that are either undefined or forward referenced (i.e. the definition occurs later on in the file), or if the expression has a value of zero, an error will be generated.
Example: 
	zmq	200	* Reserves 200 quad bytes (800 bytes)
