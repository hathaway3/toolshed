# tocgen

Table of contents generator for Sierra AGI games.

## Usage

```bash
tocgen {[<opts>]} {<infile>} {[<outfile>]} {[<opts>]}
```

## Options

- `-q`: Quiet mode.

## Description

The `tocgen` utility is a specialized tool for generating Table of Contents (TOC) files used in early Sierra On-Line AGI (Adventure Game Interpreter) games. It extracts resource entries from AGI resource files to produce a listing of game assets such as views, pictures, logic scripts, and sounds.
